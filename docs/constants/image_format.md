## 模块常量
- `IMAGE_SUFFIX_TO_FORMAT` — 文件后缀名到 PIL 格式名称的映射字典，支持以下格式：
  - JPEG: `.jpg`, `.jpeg`, `.jpe`, `.jfif`
  - JPEG2000: `.jp2`, `.j2k`, `.j2c`, `.jpc`, `.jpf`, `.jpx`
  - PNG: `.png`, `.apng`
  - BMP: `.bmp`, `.dib`
  - GIF: `.gif`
  - TIFF: `.tif`, `.tiff`
  - WEBP: `.webp`
  - HEIF: `.heic`, `.heif`
  - AVIF: `.avif`, `.avifs`
  - ICO: `.ico`
  - ICNS: `.icns`
  - TGA: `.tga`, `.icb`, `.vda`, `.vst`
  - PCX: `.pcx`
  - PPM: `.ppm`, `.pgm`, `.pbm`, `.pnm`
  - SGI: `.sgi`, `.rgb`, `.rgba`, `.bw`
  - DDS: `.dds`
  - MSP: `.msp`
  - IM: `.im`
