## 模块常量
- `user_agent_list` — 10 个常用 User-Agent 字符串列表，用于 HTTP 请求伪装。
- `zh_headers_test` — 示例中文网站 HTTP 请求头字符串。
- `GRAY`, `NAVYBLUE`, `WHITE`, `RED`, `GREEN`, `BLUE`, `YELLOW`, `ORANGE`, `PURPLE`, `CYAN`, `BLACK`, `DARKGREEN`, `DARKGRAY` — RGB 颜色元组常量。
- `COLOR_LIST` — 上述颜色常量的列表集合。
- `city_coordinate` — 中国城市名称到经纬度坐标 `[lat, lng]` 的映射字典（约 350 个城市）。
- `new_city_coordinate` — 更大范围的城市坐标字典。
